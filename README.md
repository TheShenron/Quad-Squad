## Installation


```bash
  git clone https://github.com/TheShenron/Quad-Squad-IV

  cd Quad-Squad-IV
  
  npm install 
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`MONGO_URI`

`JWT_SECRET`
